# EJERCICIO 6.1

## 1. 
Recibir un número entero por teclado y decir si es positivo.
```bash
#!/bin//bash

#Numero entero positivo

echo "introduce un numero"
read num

if [ $num -gt 0 ]
then
	echo "${num} es positivo"
else 
	echo "${num} no es positivo"
fi
```
